E-Commerce Electronics App
An Android-based e-commerce application built using Kotlin and Jetpack components, designed
specifically for buying and selling electronic products.
Features:
- Browse products
- Product details
- Cart & checkout
- Order history
Tech Stack:
- Kotlin
- MVVM Architecture
- Jetpack Components (ViewModel, LiveData, Navigation, Room)
Installation:
1. Clone repository
2. Open in Android Studio
3. Sync Gradle
4. Run app
Future Improvements:
- Payment integration
- Notifications
- Admin panel
